## Endpoints

**Implemented**

- `GET /api/git/status`
- `POST /api/git/push`
- `POST /api/git/stash`
- `GET /api/git/branches`
- `GET /api/git/log?limit=...`

**Not yet implemented**

- `POST /api/git/stage`
- `POST /api/git/unstage`
- `POST /api/git/discard`
- `POST /api/git/commit`
- `POST /api/git/pull`
- `POST /api/git/checkout`
- `POST /api/git/branches`
- `GET /api/git/commit/:sha`
